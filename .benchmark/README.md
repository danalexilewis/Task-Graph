# .benchmark Directory

The `.benchmark` directory contains all configuration, problem definitions, and scripts needed to run productivity benchmarks for TaskGraph.

## Structure

- `problems/`: Input files and definitions for benchmark scenarios.
- `scripts/`: Benchmark runner scripts and tooling.
- `results/`: Output directory for benchmark results (gitignored).

## Usage

1. Add benchmark problem definitions in `problems/`.
2. Execute the appropriate script from `scripts/` to run benchmarks.
3. Benchmark results will be saved in `results/`.

Refer to the project documentation for additional details on benchmarking best practices.
